With animations, you can change as many CSS properties as you want to, as many times you want to.

An animation is defined using keyframes.

normal - the default value, which means it plays forward from 0 % to 100%. reverse - plays the keyframe in an opposite direction from 100 % to 0% alternate - the animation first runs forward, then backward, then forward. alternate-reverse - the animation first runs backward, then forward, then backward.
